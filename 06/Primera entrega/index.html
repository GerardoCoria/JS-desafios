Tips para cebar mejores y más ricos mates

Un buen mate debe cebarse a una temperatura de 75° a 80° grados. El agua debe evitar hervir. El mate se prepara colocando la yerba a un costado, agitando para que el polvo se distribuya. Luego, colocar agua tibia para evitar que se queme. Colocar la bombilla, y no moverla.
